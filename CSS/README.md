# &#x1F6A6; CSS3 notes
+ &#x1F6A9; filter
```css
  filter: blur(px); /* 高斯模糊 */

  filter: brightness(%); /* 线性乘法 */

  filter: contrast(%); /* 对比度 */

  filter: drop-shadow(h-shadow v-shadow blur spread color); /* 阴影 */

  filter: hue-rotate(deg); /* 色相旋转 */

  filter: invert(%); /* 反转 */

  filter: grayscale(%); /* 灰度图像 */

  filter: sepia(%); /* 暖色调  褐色 */
```
