# 🔥 yarn and npm 替换淘宝镜像
  - yarn
    - 获取
      ```zsh
        yarn config get registry
      ```
    - 修改
      ```js
        yarn config set registry http://registry.npm.taobao.org/
      ```

  - npm 
    - 获取
      ```zsh
        npm get registry 
      ```
    - 修改
      ```js
        npm config set registry http://registry.npm.taobao.org/
      ```
